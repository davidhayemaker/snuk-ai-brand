## Design DNA

- Primary accent: **#3ec9c0** (teal)
- Dark circle fill: **#1a1a2e** (deep navy)
- Background: **#000000** (black)
- Text: **#ffffff** on dark, **#111111** on light
- Font: Arial Black (SNUK.AI), Arial (tagline)
- viewBox: 820×280 for horizontal lockup
